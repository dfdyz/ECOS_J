混合推进器推力分配 — 数学模型

1. 问题描述

给定一艘刚体飞船，其重心（CG）位置为 c ∈ ℝ³，装有 N 个推进器。每个推进器 i 的属性：

属性符号说明
安装位置（绝对坐标）pi ∈ ℝ³推进器在全局坐标系中的位置
原始方向di ∈ ℝ³单位向量
方向限制类型ti ∈ {STATIC, HALF_BALL, ANGLE_LIMIT}见第 2 节
最大推力Fi,max > 0

重心到推进器的力臂：ri = pi − c

需选择合适的推力矢量 vi，使得：

  • 合力 ∑ vi = Ftarget
  • 合力矩 ∑ (ri × vi) = Ttarget
  • 各推进器不超出物理限制
  • 推力总和最小

2. 推进器类型

类型符号变量约束
STATIC固定方向αi ≥ 0vi = αi · di，αi ≤ Fi,max
HALF_BALL半球vi ∈ ℝ³vi · di ≥ 0，‖vi‖ ≤ ti ≤ Fi,max
ANGLE_LIMIT锥角限制vi ∈ ℝ³vi · di ≥ ‖vi‖ cos θi，‖vi‖ ≤ ti ≤ Fi,max

2.1 STATIC（固定方向）

推力方向固定为 di，仅标量大小可调：

v_i = α_i · d_i,   α_i ≥ 0,   α_i ≤ F_i,max

LP 锥约束（2 行）：αi ≥ 0 + αi ≤ Fi,max

2.2 HALF_BALL（半球）

输出三维推力矢量 vi ∈ ℝ³，方向限制在原始朝向的半球内。

  • 半球约束：vi · di ≥ 0（与原始方向夹角 ≤ 90°）
  • 推力上限：‖vi‖ ≤ Fi,max

引入辅助变量 ti，将推力上限拆分为二阶锥约束和线性约束：

||v_i|| ≤ t_i ≤ F_i,max

2.3 ANGLE_LIMIT（锥角限制）

输出三维推力矢量 vi ∈ ℝ³，方向限制在原始朝向的锥角 θi 内。

锥角约束：vi · di ≥ ‖vi‖ · cos θi

等价于推力矢量与原始方向的夹角不超过 θi：

angle(v_i, d_i) ≤ θ_i

在 SOCP 中，锥角约束通过 G 矩阵中 ti 列的第三个非零元 cos θi 实现：

-d_i · v_i + cos(θ_i) · t_i + s = 0,   s ≥ 0

3. 二阶锥规划（SOCP）形式

3.1 决策变量

R = HALF_BALL 数，A = ANGLE_LIMIT 数，S = R + A（有 SOC 锥的推进器数），F = STATIC 数

x = [ v_0x, v_0y, v_0z, ...,        // 3S 个 v（HALF_BALL + ANGLE_LIMIT）
      t_0, ..., t_{S-1},              // S  个 t
      α_0, ..., α_{F-1} ]     // F 个 α（STATIC）

变量总数：n = 4S + F

3.2 锥结构

锥变量总数：m = 6S + 2F（S = R + A）

锥类型维度对应约束
LP 锥（HALF_BALL）2Rvi · di ≥ 0（R）+ ti ≤ Fmax（R）
LP 锥（ANGLE_LIMIT）2Avi · di ≥ ‖vi‖ cos θ（A）+ ti ≤ Fmax（A）
LP 锥（STATIC）2Fαi ≥ 0（F）+ αi ≤ Fmax（F）
SOC 锥4S（S 个，dim=4）‖vi‖ ≤ ti

3.3 约束矩阵

G 矩阵（锥约束，m × n）

行布局：

行区间维数说明
0 … S-1SHALF_BALL 半球 / ANGLE_LIMIT 锥角（vi · di ≥ 0 或 ≥ ‖vi‖ cos θ）
S … 2S-1SHALF_BALL + ANGLE_LIMIT：ti ≤ Fmax
2S … 2S+F-1FSTATIC：αj ≥ 0
2S+F … 2S+2F-1FSTATIC：αj ≤ Fmax
2S+2F … 6S+2F-14SSOC 锥，4 行 / 推进器

各变量列的非零元数：

变量类型非零数位置值
vi,x/y/zHALF_BALL / ANGLE_LIMIT2方向行 i + SOC 行-di,x/y/z, -1.0
tiHALF_BALL2上限行 + SOC 行+1.0, -1.0
tiANGLE_LIMIT3上限行 + SOC 行 + 方向行+1.0, -1.0, cos θi

A 矩阵（等式约束，6 × n）

Ax = b,   b = [F_target_x, F_target_y, F_target_z, T_target_x, T_target_y, T_target_z]^T

HALF_BALL / ANGLE_LIMIT 推进器 i（3 个变量列，每列 3 个非零元）：

列FxFyFzTxTyTz
vi,x1rz-ry
vi,y1-rzrx

ti 列全零。

STATIC 推进器 j（1 个变量列，6 个非零元）：

A[:,4S+j] = [d_jx, d_jy, d_jz,
              r_jy·d_jz - r_jz·d_jy,
              r_jz·d_jx - r_jx·d_jz,
              r_jx·d_jy - r_jy·d_jx]^T

c = [0 … 0 (3S),  1 … 1 (S+F)]^T

4. 重心变化

推进器位置 pi（绝对坐标）和重心 c（绝对坐标）由外部更新。每次求解时力臂实时计算：

r_i = p_i - c

力臂 ri 仅出现在 A 矩阵的力矩行（A3:6,:）中。对于 HALF_BALL / ANGLE_LIMIT 推进器，力矩行使用 ri × vi（vi 为求解变量）；对于 STATIC 推进器，力矩行使用 ri × di（di 为固定方向）。

5. 方向变化

推进器方向 di 可在运行时更新，影响：

影响范围HALF_BALL / ANGLE_LIMITSTATIC
G 矩阵方向行系数 -di,x/y/z无影响
A 矩阵无影响（方向由求解器决定）全部 6 个条目（力 + 力矩）

对于 ANGLE_LIMIT 推进器，方向变化时还需更新 ti 列中 cos θi 的值。

6. 最大推力变化

Fi,max 可运行时修改，影响 G 矩阵中上限行对应的 h 向量值（hS+i 或 h2S+F+j）。
